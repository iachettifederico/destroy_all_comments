* Destroy All Comments

Destroy all the commented lines on your project.

** Installation

Add this line to your application's Gemfile:

#+BEGIN_SRC
gem 'destroy_all_comments'
#+END_SRC

And then execute:

#+BEGIN_SRC
$ bundle
#+END_SRC

Or install it yourself as:
#+BEGIN_SRC
$ gem install destroy_all_comments
#+END_SRC
